マーク基準

  • マーカーを配置する際の基準とするものを選びます。
  • ノート始端: 各ノートの先頭に合わせてマーカーを配置します。
    • 「ノートの終端をマーク」をチェックした場合、各ノートの終端から
      「後方余白」だけ後ろに切り捨て用のマーカーを設置します。
    • ノート終端から次のノートまでの間隔が「後方余白」以下の場合は
      切り捨てマーカーは設置されません。
  • マーカーイベント: メタタイプが「Marker」である
    メタテキストイベントの位置にマーカーを配置します。
    • 複数のノートを一つにまとめたい場合はこちらを利用します。
    • MIDIモード/MIDI整理が出力するMIDIファイルは
      このタイプに適した形式になっています。
