スライス

読み込んだ音声ファイルを、BMS風のUIを使って分割する機能です。

  • 音声の波形と小節線を確認しながら、ノートの配置によってスライス位置を設定できます。
    • ノート配置の感覚は、恐らくBmsONEにある程度近いかと思われます。
      ただし、このアプリはVer 0.4 時点で.bmsonの編集には対応していません。
  • スライスした音声ファイルを分割保存できます。
    • 分割保存したそのデータをそのまま定義&配置することもできます。
  • MIDIファイルを基準にスライスしたい場合は、Waveモード/トラックマークでマーカーを配置したうえで保存し、そのファイルを読み込んでください。
    • 保存先のフォルダを適切に選べば、Waveモード/スライスで分割保存したファイルをそのまま定義&配置に利用できます。

説明

要素をクリックすると説明が表示されます。
  • 「機能」タブまたはショートカットキー(Ctrl+8)からこのウィンドウを開く場合、その前にスライスする音声ファイルを選ぶ必要がありますが、
    以下の状態では自動的に音声ファイルが読み込まれた状態になります。
    • 譜面表示エリアで、実際に音声ファイルが存在する定義番号の音声ノートが1個だけ選択されている
    • 上記以外で、#WAV定義リストがアクティブ(選択項目が青っぽくなる)かつ実際に音声ファイルが存在する定義が選択されている
  • マウスホイールを回すことで縦(時間)方向にスクロールします。
    • Ctrlを押している場合は縦スケールを変更します。
  • ノートを選択/追加した後にカーソルキーの↑↓を押すと、その直後/直前のノートを選択します。
    • この際、「クリックで再生」がチェックされていれば、選択されたノートの内容を再生します。
