定義

プレイ時に参照される音声や映像、及び各小節の長さについての定義リストです。

説明

要素をクリックすると説明が表示されます。
  • 定義番号が共有されるもの(#BMPと#BGAなど)は、それぞれ1つのリストに統合されています。
    • 定義の内容に特殊プレフィックスを付加することで、そのタイプの定義であると解釈されます。
  • (各リスト共通事項)
    • 項目上で右クリックすることで、コンテキストメニューを開きます。
      • 内容はタブによって異なるので、それぞれの説明を参照してください。
    • 降下中に親で定義されている項目は、その値が薄く表示されます。

特殊プレフィックス

  • 定義の内容を特定の文字列から始めることで、定義ヘッダ(#WAVの部分)の種類を変えることができます。
  • #WAVの特殊プレフィックス
    • <EXWAV>
      • 定義ヘッダを#EXWAVに変換します。(参考)
        　変換前(エディタ上) → 　変換後(保存時) #WAV01 <EXWAV> pvf pan volume freq filename → #EXWAV01 pvf pan volume freq filename
  • #BMPの特殊プレフィックス
    • <BGA>
      • 定義ヘッダを#BGAに変換します。(参考)
        　変換前(エディタ上) → 　変換後(保存時) #BMP02 <BGA> id x1 y1 x2 y2 dx dy → #BGA02 id x1 y1 x2 y2 dx dy
    • <@BGA>
      • 定義ヘッダを#@BGAに変換します。(参考)
        　変換前(エディタ上) → 　変換後(保存時) #BMP03 <@BGA> id sx sy w h dx dy → #@BGA03 id sx sy w h dx dy
    • <EXBMP>
      • 定義ヘッダを#EXBMPに変換します。(参考)
        　変換前(エディタ上) → 　変換後(保存時) #BMP04 <EXBMP> a,r,g,b filename → #EXBMP04 a,r,g,b filename
