開始位置

  • 音声の開始位置を指定します。
    • ノート: 「対象」のうち最初のノートの位置が音声の開始位置となります。
      • そのノートに定義された音声自体に先頭の空白がある場合は、
        書き出される音声の先頭にも空白が生じることになります。
    • #000: 譜面の開始位置が音声の開始位置となります。
      • 最初のノートが小節#000の先頭に配置されていない限り、
        書き出される音声の先頭には空白が生じることになります。
      • 動画用に書き出すなど、タイミングが一定である必要がある場合は
        こちらを選択してください。
