譜面タイプ

  • このスタイルが表す譜面のタイプです。
  • 左側
    • ファイルの拡張子に影響する譜面タイプ。
    • タイプが「Popn」のスタイルを選択している場合、
      「名前を付けて保存」のデフォルト拡張子が.pmsになります。
      • 「上書き保存」の場合は拡張子は変更されません。
    • .pmsファイルを開いた場合、
      タイプが「Popn」となっているスタイルに自動的に切り替わります。
  • 右側
    • #PLAYERヘッダーに影響するプレイヤー数。
    • タイプが「Single」または「Double」のスタイルを選択している場合、
      BMSの#PLAYERヘッダーがその値になります。
    • ファイルを開く際には実際の使用レーン数のみを参照し、
      #PLAYERによってスタイルが切り替わることはありません。
