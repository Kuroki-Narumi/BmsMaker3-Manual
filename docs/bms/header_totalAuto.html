#TOTALの自動入力

  • 譜面のノート数を使い、計算式に基づいて#TOTALの値を自動入力します。
  • を押すと計算式を確認/編集できます。
    • 詳細は下記「#TOTAL自動入力の数式の詳細」を確認してください。
    • テキストボックスの入力内容はアプリ設定に記憶され、
      起動するたびに前回の入力内容が復元されます。
    • テキストボックスの下のボタンを押すとプリセットが入力されます。
      • Beat: floor(max(260, 760.5 * n / (n + 650)))
      • Popn: floor(n <= 0 ? 0 : n * floor(3072 / n) / 10.24)
    • を押すと入力中の数式をプリセットとして保存します。
    • 保存済みのプリセットは下に一覧表示されます。
      • を押すとそのプリセットを適用します。
      • を押すとそのプリセットを削除します。
