追加モード

F3
  • 編集モードを「追加」に切り替えます。
  • 譜面表示エリアで左クリックした場合、その位置に新たなノートを配置します。
    • 追加されたノートは選択状態になります。
    • クリック位置に既にノートが存在する場合、元々のノートは削除され、
      上書きされます。
    • 定義番号00が選択されている場合は、クリックした位置のノートを削除します。
  • 演奏レーン上にカーソルがある時、
    以下のキーを押している間、 配置予定のノートの属性を変更します。
  • 地雷及び指揮者レーンに対するノート配置の際には、
    その値を入力するウィンドウが表示されます。
    • 入力をキャンセルした場合、ノートは配置されません。
    • 0を入力し、その位置に既にノートが存在する場合、
      そのノートは削除され、新たに配置はされません。
  • 譜面表示エリアでミドル(ホイール)クリックした場合、
    その位置を始点として連続配置ウィンドウを開きます。
