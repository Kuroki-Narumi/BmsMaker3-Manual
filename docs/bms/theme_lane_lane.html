レーン番号

  • このレーンに表示するノートのレーン番号です。
  • 1から71までの値を受け付けます。
  • 同じ番号を指定したレーンが複数ある場合、
    ノートはそのうちいずれか1つにしか表示されません。
    • 全てのノートが1つのレーンに表示されますが、
      どのレーンに表示されるかは不定です。
  • 歴史的経緯により、このレーン番号は見た目通りの順番にはなっていません。
    • 詳しい配置はBMS command memoなどを参照してください。(※)
    • ※上記ページのチャンネル番号表記は以下のように読み替えてください。
      • 1x台→xのみに置き換える
      • 2x台→x + 36に置き換える
      • 例えば「BMSの2Pスクラッチ(ch.26)」のレーン番号は、
        BmsMaker3では6 + 36 = 42です。
