#RANK

  • この譜面の判定の厳しさを表す値です。
    • 難度を表す数値ではありません。
  • 以下の4段階です。
    1. VeryHard: 最も厳しい判定基準です。
      • 多くのBMSプレイヤーでは、最高判定(P.GREAT)の幅が
        1か2フレームになります。
    2. Hard: 厳しい判定基準です。
    3. Normal: 通常の判定基準です。
    4. Easy: 最も緩い判定基準です。
      • ノート数の多い譜面は、リズムの正確さよりノートを取りこぼさない
        ことが求められるため、この判定にするのが無難です。
  • 特別な理由が無い限りはNormalかEasyを選ぶべきです。
