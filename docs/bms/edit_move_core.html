• 譜面表示エリアでノートが存在しない領域を左クリックした場合、範囲選択を開始します。
  • 譜面表示エリアでノートを左クリックした場合、そのノートを選択します。
    • 同時に押すキーによって挙動が変化します。
      • 下記以外: 既存の選択を解除し、対象のノートのみを選択します。
      • ShiftまたはCtrl: 既存の選択に対象のノートを追加します。
      • Alt: 対象のノートの選択状態を切り替えます。
        • 対象のノートが選択済みならそのノートのみ選択状態を解除し、
          そうでなければ既存の選択に対象のノートを追加します。
