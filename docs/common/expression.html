• 定数値(整数、小数)
  • 四則演算子(+-*/%)
  • 条件演算子
    • 以下の表現は計算時に、それぞれ右側で示された値に置き換わります。
    • x & y: (論理積)xが0なら0、そうでなければy
    • x | y: (論理和)xが0ならy、そうでなければx
    • x ^ y: (排他的論理和)xが0ならy、yが0ならx、それ以外の場合は0
    • [条件] ? x : y: (条件分岐)[条件]が0以外ならx、そうでなければy
    • x = y: xとyが等しい (等しければ1、そうでなければ0)
    • x != y: xとyが等しくない
    • x <= y: xがy以下
      • x < yx >= yx > yも同様。
  • 関数
    • 以下の表現は計算時に、それぞれ右側で示された値に置き換わります。
      • 大文字と小文字は区別されます。下記の関数は全て小文字で記述する必要があります。
      • 括弧も含めて記述する必要があります。
      • 括弧内の要素(引数)は,で区切り、その数(max()なら2個)は関数ごとに正確に一致しなければなりません。
    • max(x, y): xとyの大きい方
    • min(x, y): xとyの小さい方
    • floor(x): x以下の最大の整数
    • ceil(x): x以上の最小の整数
    • trunc(x): xの整数部分
    • round(x, y): xを小数点以下y桁で丸めたもの
